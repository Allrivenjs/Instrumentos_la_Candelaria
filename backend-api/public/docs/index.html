Introduction

Api template for ecommer's basic

This documentation aims to provide all the information you need to work with our API.

This api is written in PHP-laravel, and it is based on the structure of a basic emmers for its operation, I hope it will be very helpful :D

Base URL

http://jaime.registrodocumental.com/Instrumentos_la_Candelaria/backend-api/public

Authenticating requests

Authenticate requests to this API's endpoints by sending an Authorization header with the value "Bearer your-token".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.
