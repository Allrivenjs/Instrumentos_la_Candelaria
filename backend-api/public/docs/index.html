Introduction

Api template for ecommer's basic

This documentation aims to provide all the information you need to work with our API.

This api is written in PHP-laravel, and it is based on the structure of a basic emmers for its operation, I hope it will be very helpful :D

Base URL

http://jaime.registrodocumental.com/Instrumentos_la_Candelaria/backend-api/public

Authenticating requests

This API is authenticated by sending an Authorization header with the value "Bearer your-token".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Login Auth

Method to authenticate yourself in the api, it is necessary to use some of its functions
Example request:
curl --request POST \
    "http://localhost:8000/api/auth/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"\\\"Julian\\\"\",
    \"password\": \"\\\"passwordsecret\\\"\"
}"

const url = new URL(
    "http://localhost:8000/api/auth/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "\"Julian\"",
    "password": "\"passwordsecret\""
};

    method: "POST",
    headers,
    body: JSON.stringify(body),
